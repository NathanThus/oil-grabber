# Oilgrabber

A simple plugin that gets all of the data about the various enhancements in the game [Sulfur](https://store.steampowered.com/app/2124120/SULFUR/), and exports it to JSON for analysis. This project was started as a way of helping out the community, as we noticed the data from the oils on the wiki was out of date.

## Requirements
Bepinex 5.6.X

Sulfur - [Available at](https://store.steampowered.com/app/2124120/SULFUR/)

## Usage

Simply put the plugin into the plugins folder and run the game, by the time the game has made it to the start screen, the plugin is finished.
